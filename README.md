# Sistema de Análise de Jogadores de Futebol

Projeto desenvolvido em Java com POO e MySQL.

## Funcionalidades
- Cadastro de jogadores
- Separação por posição
- Estatísticas por atleta
- Cálculo automático de nota de desempenho
- Integração com banco de dados

## Tecnologias
- Java
- MySQL
- NetBeans

## Estrutura
- Jogador (classe base)
- Atacante
- Meio-campo
- Defensor
- Goleiro

## Objetivo
Simular um sistema de scouting esportivo baseado em dados reais.
